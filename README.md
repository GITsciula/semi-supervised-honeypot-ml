# Detekcja i klasyfikacja ataków na podstawie danych z honeypot z wykorzystaniem sztucznej inteligencji.

Celem niniejszego repozytorium jest przedstawienie implementacji badań prowadzonych w ramach pracy magisterskiej, dotyczących wykorzystania współczesnych narzędzi uczenia maszynowego i technik analizy danych do badania informacji pochodzących z systemów typu honeypot. Szczególny nacisk położono na wykrywanie oraz klasyfikację wzorców ataków.
